🐝 Overview

⚙️ Setup & Installation

Prerequisites

⚠️ Requirements:
  • Node.js 18+ with npm or yarn
  • Claude Code CLI installed and configured
  • MCP (Model Context Protocol) server support
  • Minimum 4GB RAM for agent coordination

Installation Steps

# Install Claude Flow framework npm install -g claude-flow@alpha # Verify installation npx claude-flow version # Initialize MCP server claude mcp add claude-flow npx claude-flow@alpha mcp start

Configuration

Set up your environment variables and configuration files:

2️⃣ Execution Phase

# After every step npx claude-flow@alpha hooks post-edit --file "[file]" --memory-key "swarm/[agent]/[step]" # Decision notifications npx claude-flow@alpha hooks notify --message "[decision]"

# Task completion npx claude-flow@alpha hooks post-task --task-id "[task]" --analyze-performance true # Session cleanup npx claude-flow@alpha hooks session-end --export-metrics true

🔧 Troubleshooting

❌ Agent Spawn Failures +

Symptoms: Agents fail to initialize or respond

Common Causes:

  • Insufficient memory allocation (< 4GB)
  • MCP server not properly configured
  • Network connectivity issues
  • Agent limit exceeded (max 54)

Solutions:

# Check agent status npx claude-flow agent status # Restart MCP server claude mcp restart claude-flow # Verify memory availability free -h # Check agent limits npx claude-flow config show
🔄 Communication Failures +

Symptoms: Agents not communicating or messages lost

Solutions:

# Check message queue status npx claude-flow queue status # Reset communication channels npx claude-flow reset-channels # Verify network connectivity npx claude-flow network-test # Enable debug logging export DEBUG=claude-flow:*
⚡ Performance Issues +

Symptoms: Slow response times or high resource usage

Optimization Strategies:

  • Reduce concurrent agent count
  • Enable message batching
  • Implement connection pooling
  • Use appropriate coordination topology
{ "performance": { "maxConcurrentAgents": 8, "messageBatching": true, "connectionPool": 50, "topology": "hierarchical" } }
🗄️ Memory & State Issues +

Symptoms: State inconsistencies or memory leaks

Solutions:

# Clear shared memory npx claude-flow memory clear # Check memory usage npx claude-flow memory stats # Validate state consistency npx claude-flow state validate # Garbage collect npx claude-flow gc --force
🚨 Emergency Recovery:
If your swarm becomes unresponsive, use the emergency recovery commands:
# Emergency stop all agents npx claude-flow emergency-stop # Reset entire swarm npx claude-flow reset --hard # Restart with clean state npx claude-flow restart --clean
